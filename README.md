# MacData Calendar

Нативная утилита для macOS: компактная иконка календаря в menu bar открывает сетку месяца — быстро узнать **день недели** для любой даты, не открывая Calendar.app.

Системные часы Apple **не трогаем** — MacData живёт рядом, как отдельная кнопка.

| | |
|---|---|
| **Платформа** | macOS 14+ (Sonoma и новее) |
| **Стек** | Swift 5.9 · SwiftUI · MenuBarExtra |
| **Лицензия** | [MIT](LICENSE) |

## Возможности

- **Сетка месяца** — понедельник первым, навигация ‹ › и «Сегодня»
- **Выходные и праздники РФ** — красная семантика + производственный календарь 2025–2026
- **Полоска подсказки** — при наведении: `понедельник, 15 июня · Выходной`
- **Копирование даты** — клик по дню → буфер обмена + «Скопировано»
- **Клавиатура** — ← → месяц, **T** — текущий месяц
- **Автозапуск** — «Запускать при входе» в footer popover
- **Выход** — ссылка в popover или ПКМ по иконке в menu bar

## Установка

### Готовый билд (Releases)

1. Скачай `MacDataCalendar-1.0.0.zip` из [Releases](https://github.com/OTumanov/macData/releases).
2. Распакуй и перетащи **MacDataCalendar.app** в `/Applications`.
3. Открой приложение — иконка с числом появится в menu bar.

При первом запуске macOS может запросить разрешение — это нормально для ad-hoc подписанного приложения.

### Сборка из исходников

```bash
git clone git@github.com:OTumanov/macData.git
cd macData
make          # → build/MacDataCalendar.app
make run      # собрать и открыть
cp -R build/MacDataCalendar.app /Applications/
```

Релизный архив:

```bash
make release  # → dist/MacDataCalendar-1.0.0.zip
```

Тесты (нужен полный Xcode):

```bash
make test
```

## Использование

```
        ‹   Июнь 2026   ›
  Пн  Вт  Ср  Чт  Пт  Сб  Вс
  ┌─────────────────────────┐
  │  воскресенье, 15 июня    │  ← полоска при наведении
  └─────────────────────────┘
   1   2   3   4   5   6   7
  ...
─────────────────────────────
☑ Запускать при входе   Выйти
```

| Действие | Как |
|----------|-----|
| Открыть календарь | Клик по иконке в menu bar |
| Предыдущий / следующий месяц | ‹ › или ← → |
| Вернуться к сегодня | «Сегодня» или **T** |
| Узнать день недели | Навести на дату |
| Скопировать дату | Клик по дню |
| Выйти | «Выйти» или ПКМ по иконке |
